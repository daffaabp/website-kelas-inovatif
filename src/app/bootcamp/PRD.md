# Product Requirement Document (PRD)
## Bootcamp Masterclass Penulisan Akademik

### 1. Ringkasan Proyek
Landing page untuk **Bootcamp Masterclass Penulisan Akademik**, sebuah program pelatihan intensif "End-to-End" yang merangkum seluruh proses produksi karya ilmiah. Program ini memadukan keahlian manusia (narasumber ahli) dengan efisiensi teknologi (AI Tools) untuk membantu akademisi mempublikasikan riset mereka di level nasional (Sinta) maupun internasional (Scopus).

### 2. Tujuan (Objectives)
*   **Menampilkan Integrasi Materi**: Menegaskan bahwa peserta tidak belajar secara terpotong-potong, melainkan dalam satu aliran proses (Pipeline) yang utuh.
*   **Lead Generation**: Mengarahkan peminat serius untuk menghubungi Admin (Chat/Consultation) guna mendapatkan konversi personal.
*   **Posisi Premium**: Membangun citra "Masterclass" yang eksklusif dan berbobot.

### 3. Target Audiens
*   Dosen yang mengejar KUM/Beban Kerja Dosen.
*   Mahasiswa S2/S3 yang sedang menyusun Tesis/Disertasi.
*   Peneliti/Praktisi yang ingin menembus publikasi bereputasi.

### 4. Fitur & Struktur Halaman

#### A. Hero Section (Impression)
*   **Headline**: "Bootcamp Masterclass: Dari Ide hingga Publikasi Ilmiah".
*   **Sub-headline**: "Rangkaian pelatihan intensif 4-in-1: Metodologi, AI Writing, Olah Data, hingga Tembus Jurnal Scopus/Sinta."
*   **CTA Utama**: Tombol "Hubungi Admin" / "Konsultasi Program" (Link ke WhatsApp).

#### B. The Roadmap (Inti Konten)
Visualisasi alur perjalanan peserta (Journey) yang saling bersambung. Tidak boleh terlihat sebagai kotak-kotak materi terpisah.
1.  **Metodologi Penelitian** (Foundation)
    *   *Key Point*: Ketepatan desain riset adalah kunci diterima jurnal.
2.  **Academic Writing with AI** (Acceleration)
    *   *Key Point*: Teknik prompting, paraphrasing, dan citasi otomatis untuk mempercepat drafting naskah.
3.  **Data Analysis** (Evidence)
    *   *Key Point*: Tutorial olah data Kuantitatif (SPSS/SEM-PLS) atau Kualitatif (NVivo/Atlas.ti).
4.  **Publication Strategy** (Goal)
    *   *Key Point*: Bedah jurnal, korespondensi dengan editor, dan strategi revisi mayor/minor.

#### C. Mengapa Bootcamp Ini Berbeda? (Unique Selling Point)
*   **Integrated Workflow**: Satu kali daftar, dapat ilmu hulu ke hilir.
*   **Outcome Oriented**: Fokus pada penyelesaian naskah, bukan sekadar teori.
*   **Expert Mentors**: Profil tim narasumber Kelas Inovatif.

#### D. Profil Narasumber
*   Section khusus menampilkan foto dan short bio para pakar.

#### E. Call to Action (Floating/Sticky)
*   Mengingat ini program premium, pendekatan pendaftarannya lewat jalur personal.
*   Tombol: "Tertarik Mengikuti? Chat Admin Sekarang".

### 5. Catatan Teknis
*   **Framework**: Next.js (React).
*   **Styling**: Tailwind CSS (Gunakan komponen `Timeline` atau `Step Process` untuk Roadmap).
*   **Assets**: Gunakan ikon modern untuk merepresentasikan 4 pilar materi (e.g., Ikon Mikroskop untuk Metode, Ikon Robot/Brain untuk AI, Ikon Grafik untuk Data, Ikon Dokumen/Globe untuk Publikasi).
*   **Interaksi**: Hover effects pada poin roadmap untuk menampilkan detail kurikulum.

### 6. Rencana Pengembangan (Future)
*   Halaman ini didesain fleksibel agar tanggal dan batch bisa diupdate manual atau via CMS di kemudian hari tanpa merombak struktur layout.
